# OpenCV Frame Extractor

## Description
This project extracts frames from videos using OpenCV.

## Requirements
- Python 3.x
- OpenCV

## Installation
```bash
pip install opencv-contrib-python
```

## Usage
Extract frames:
```bash
python reading_and_saving_video_frames.py
```

Play frames:
```bash
python framesPlay.py
```

## Controls
Press 'q' to stop video playback
